Script d’Inventaire Collaborateurs

Automatisation & Reporting – Équipe IT

Présentation

Ce script PowerShell collecte automatiquement les informations essentielles d’un poste : système, matériel, logiciels, utilisateur, réseau, et met à jour la base de données d’inventaire sur le serveur de collecte.

Téléchargement

Cliquez ci-dessous pour télécharger la dernière version du script :

Télécharger le script PowerShell

Exemple d’exécution

powershell.exe -ExecutionPolicy Bypass -File .\inventaire.ps1

Fonctionnalités principales

  • Collecte OS, build, CPU, RAM, IP, numéro de série
  • Analyse des logiciels installés (versions les plus récentes)
  • Détection du dernier logon et dernier boot
  • Export TXT détaillé + export CSV global
  • Rangement automatique dans les bons dossiers réseau
